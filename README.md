## Comandos

---

### Instalar dependências

```bash
npm install
```

### Rodar localmente

```bash
npm start
```

### Rodar o storybook localmente

```bash
npm run storybook
```

### Build

```bash
npm run build
```

### Build do storybook

```bash
npm run build-storybook
```

## Estrutura do projeto
